# happy-copy

#### 介绍
嗨皮拷贝是一个桌面端办公效率工具，旨在帮助用户快速拷贝指定内容，与输入法的用户自定义短语功能有异曲同工之妙，但又不尽相同，适用场景和内容不通。

#### 软件架构
开发平台：electron
开发框架：vuejs
组件库：elementUI
样式库：tailwindcss


#### 安装教程

1. 下载源代码：git clone https://gitee.com/enobj/happy-copy.git
2. 安装依赖：npm run install
3. 启动开发环境：npm run start
4. 打包：npm run make
