# happy-copy

#### 介绍
嗨皮拷贝是一个桌面端办公效率工具，旨在帮助用户快速拷贝指定内容。

#### 特性
1. 常驻托盘，一键复制
2. 支持隐藏敏感内容
3. 支持图片项
4. ctrl+v快速唤起新建项
5. 拖拽排序

#### 软件架构
- 开发平台：electron
- 开发框架：vuejs
- 组件库：elementUI
- 样式库：tailwindcss

1. 下载源代码：git clone https://gitlab.ctbiyi.com/jienhui/happy-copy.git

3. 启动开发环境：npm run start
4. 打包：npm run make
